# Laravel + TypeScript + React
Boilerplate code featuring Laravel 5.8, TypeScript 3.3, React 16.8.

## Also Starring:
- ESLint (extending AirBNB and @typescript-eslint)
- Redux (with Thunk and DevTools)
- React Router DOM
- React Transition Group
- CSSTransitions
- FontAwesome 5
- StyledComponents
- normalize.css
- BrowserSync
- Axios
